# speechy
a simple text-to-speech made with python

# How to download it:

```sh
sudo apt-get install git
git clone https://github.com/NacreousDawn596/speechy
cd speechy
chmod u+x setup.sh
./setup.sh
```

now you can launch it by writing *speechy* after reopening the terminal, enjoy! ☺️
